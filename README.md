# PrintQuote Pro

Dynamic Print Quote ERP for print vendors. Create quotes, save presets, manage quality per category.

## Features

- **Service Categories:** Banner, Brochure, Poster, Pamphlet, Apparel, Sticker, Business Card
- **Predefined Sizes:** per category (A4, A3, 6Ã—2 ft, etc.) with unit selector (in, cm, ft, mm)
- **Quality Options:** Per-category quality with price; set in Settings
- **Pricing:** Per Unit, Per Sq. Ft, Per Sq. Inch, Bulk Quantity
- **Labor, Die-Cut, Rush Fee:** Configurable in Settings
- **Client Quote:** Copy clean quote with specs, file format, aspect ratio
- **Languages:** English, Tamil, Tanglish

## Run

Open `index.html` in a browser or serve with any static server:

```bash
python -m http.server 8080
```

Then visit http://localhost:8080

## Tech

- HTML5, Tailwind CSS (CDN), Vanilla JavaScript
- No build required
